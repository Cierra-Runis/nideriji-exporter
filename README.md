# 📚 简介

本脚本使用 你的日记 API 进行日记与图片的抓取，并导出 json 格式的日记和 json 格式与 jpg 格式的图片

# 📦 使用方式

使用 `git clone https://github.com/Cierra-Runis/nideriji-exporter.git` 指令拉取本仓库至本地，运行 `nideriji_export.py` 并安指示输入帐号密码即可
